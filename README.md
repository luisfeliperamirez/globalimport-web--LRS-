Descripción del Proyecto
Se desarrollará un sitio web de comercio electrónico enfocado en una empresa importadora de productos.
El objetivo es integrar el uso de HTML y JavaScript aplicados a un caso real, practicando buenas 
prácticas de desarrollo y control de versiones desde el inicio del proyecto.

Contexto del Negocio
La empresa ficticia "GlobalImport S.A." se dedica a la importación y distribución de productos 
electrónicos, indumentaria y artículos del hogar. El sitio web debe transmitir confianza, 
profesionalismo y facilitar el contacto con potenciales clientes
